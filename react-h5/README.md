## 目录结构
```shell
project-root/
  |- src/                           # Source code
  |   |- assets/                   # Static assets (images, fonts, etc.)
  |   |- styles/                   # CSS or pre-processor styles (e.g., Sass, Less)
  |   |- scripts/                  # JavaScript files
  |   |- views/                    # HTML or template files
  |   |- index.html                # Main HTML file
  |- public/                       # Public files (not processed by build tools)
  |   |- images/                   # Public images
  |   |- fonts/                    # Public fonts
  |   |- ...
  |- dist/                         # Build output (generated by build tools)
  |- node_modules/                 # Node.js dependencies (generated by npm or yarn)
  |- .gitignore                    # Git ignore file
  |- package.json                  # Node.js dependencies and scripts
  |- webpack.config.js             # Webpack configuration file (if using Webpack)
  |- README.md                     # Project documentation

```
